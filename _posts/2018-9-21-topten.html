A NodeJS command line package for scraping the 10 most read stories from the BBC website to your terminal window

The node shell script runs from the command line and scrapes the ten most read articles from the BBC website into the terminal window, simple and effective, a great way to check the news, perfect to pipe to less. The code for this project is available on GitHub.
