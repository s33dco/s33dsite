Banco

A bank statement analyizer run from the command line or bundled into your rails app. Banco imports a .CSV file and outputs a summary of the transactions in an easy to understand format. Written in Ruby with no third party dependencies.

Available as a Ruby Gem or you can clone the source code from GitHub. Open Source and free, (as in speech and beer), to use.
